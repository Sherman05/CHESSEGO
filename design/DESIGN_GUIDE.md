# DESIGN GUIDE — chess-T1
# ОБЯЗАТЕЛЬНО К ПРОЧТЕНИЮ ПЕРЕД НАПИСАНИЕМ ЛЮБОГО UI-КОДА

## КАК ИСПОЛЬЗОВАТЬ ЭТОТ ГАЙД

1. **СНАЧАЛА** открой и рассмотри файлы `mockup_extended-1.png` и `mockup_intro-1.png` — это эталонные макеты от заказчика
2. Каждый UI-элемент ниже привязан к конкретной области на макете
3. **ЦВЕТА** бери ТОЛЬКО из этого гайда — не придумывай свои
4. **ФИГУРЫ** — из файла `pieces_all.svg` (Inkscape SVG, Layer 2 содержит 14 групп = 14 фигур)
5. **КНОПКИ** — из файла `buttons_all.svg` (Inkscape SVG, аналогичная структура)

> **ВАЖНО:** SVG-файлы от заказчика сделаны в Inkscape с нестандартными координатами.
> Для извлечения отдельных фигур используй Inkscape CLI:
> ```bash
> inkscape pieces_all.svg --export-id=GROUP_ID --export-id-only --export-type=png --export-filename=piece_name.png
> ```
> Список group ID см. ниже в разделе ФИГУРЫ.
>
> **ЕСЛИ Inkscape недоступен** — нарисуй фигуры программно (SVG path) по описанию ниже.
> Фигуры ПРОСТЫЕ — это геометрические формы (пятиугольники, трапеции, треугольники), а не детализированные шахматные фигуры.

### Пропорции (из макета)
- Доска занимает ~70% ширины окна
- Кассы фигур (левая + правая) — по ~8% ширины каждая
- Нотация (буквы/цифры) — между кассой и доской
- Верхняя панель кнопок — ~40px высота
- Нижняя панель кнопок — ~40px высота
- Клетка доски — квадратная, размер = (высота окна - панели - нотация) / 8

## ФИГУРЫ — ОПИСАНИЕ ВНЕШНЕГО ВИДА

Файл: `pieces_all.svg`, Layer 2 содержит 14 групп.
Group IDs (для Inkscape export): g8188, g8158, g8202, g8125, g8168, g8189, g8215, g1011, g8199, g969, g997, g962, g8429, g8462

### Общий стиль всех фигур
- Контур: тёмно-коричневый (#411B1A), толщина 2-3px
- Белые фигуры: заливка #FFFFFF (белый), контур #411B1A
- Чёрные фигуры: заливка #000000 (чёрный), контур #411B1A
- Все фигуры — **геометрические**, НЕ классические шахматные
- Высота иконки = 90% от стороны клетки, центрируется, отступ от низа клетки

### Описание каждой фигуры (для программной отрисовки если SVG не извлекается)

**Король (Кр)** — Пятиугольник (home plate shape): трапеция с треугольником сверху. Сверху маленький кружок (навершие). Внутри вертикальная линия по центру.

**Коннет (Кт)** — Самая крупная фигура. Пятиугольник похожий на Короля, но шире и выше. Внутри три вертикальные линии (|||). Сверху навершие — кружок с перечёркнутым крестиком.

**Принц (Пр)** — Пятиугольник средний. Внутри одна вертикальная линия. Сверху навершие — маленький кружок.

**Риттер (Рт)** — Пятиугольник средний. Внутри две вертикальные линии (||). Без навершия сверху.

**Кнехт (Кн)** — Самая маленькая фигура. Простая трапеция (равнобедренная, сужается кверху). Без навершия, без линий внутри.

**Вер Кнехт (ВК)** — Чуть крупнее Кнехта. Трапеция + маленький треугольник навершие сверху. Внутри одна вертикальная линия.

**Разведчик (Рк)** — Пятиугольник. Внутри одна вертикальная полоса. Сверху навершие — маленький кружок. Отличается от Принца формой основания.

### Если Inkscape доступен — команды для нарезки:
```bash
# Для каждого group ID из Layer 2:
GROUPS="g8188 g8158 g8202 g8125 g8168 g8189 g8215 g1011 g8199 g969 g997 g962 g8429 g8462"
for GID in $GROUPS; do
  inkscape pieces_all.svg \
    --export-id=$GID \
    --export-id-only \
    --export-type=png \
    --export-width=200 \
    --export-filename=${GID}.png
done
# Затем визуально определи какая фигура какая и переименуй
```

### Если Inkscape НЕ доступен — нарисуй SVG программно:
Каждая фигура — это простой SVG path. Вот базовые формы:

```svg
<!-- Кнехт (самая простая — трапеция) -->
<svg viewBox="0 0 60 70">
  <path d="M15,65 L10,25 L30,5 L50,25 L45,65 Z" 
        fill="WHITE_OR_BLACK" stroke="#411B1A" stroke-width="2"/>
</svg>

<!-- Принц (пятиугольник + навершие + одна линия) -->
<svg viewBox="0 0 60 80">
  <circle cx="30" cy="5" r="4" fill="WHITE_OR_BLACK" stroke="#411B1A" stroke-width="2"/>
  <path d="M15,72 L10,30 L30,12 L50,30 L45,72 Z" 
        fill="WHITE_OR_BLACK" stroke="#411B1A" stroke-width="2"/>
  <line x1="30" y1="25" x2="30" y2="65" stroke="#411B1A" stroke-width="2" opacity="0.5"/>
</svg>

<!-- Коннет (широкий + 3 линии + навершие с крестом) -->
<svg viewBox="0 0 70 85">
  <circle cx="35" cy="5" r="5" fill="WHITE_OR_BLACK" stroke="#411B1A" stroke-width="2"/>
  <line x1="31" y1="2" x2="39" y2="8" stroke="#411B1A" stroke-width="1.5"/>
  <line x1="39" y1="2" x2="31" y2="8" stroke="#411B1A" stroke-width="1.5"/>
  <path d="M12,78 L8,32 L35,12 L62,32 L58,78 Z" 
        fill="WHITE_OR_BLACK" stroke="#411B1A" stroke-width="2"/>
  <line x1="28" y1="25" x2="28" y2="70" stroke="CONTRAST" stroke-width="2"/>
  <line x1="35" y1="25" x2="35" y2="70" stroke="CONTRAST" stroke-width="2"/>
  <line x1="42" y1="25" x2="42" y2="70" stroke="CONTRAST" stroke-width="2"/>
</svg>
```
CONTRAST = для белых фигур используй #411B1A (тёмные линии), для чёрных — #FFFEF7 (светлые линии)

## КНОПКИ УПРАВЛЕНИЯ — ОПИСАНИЕ

Файл: `buttons_all.svg` (аналогичная структура, Inkscape).

### Верхний ряд (слева направо на макете):
1. **«Начальная расстановка»** — иконка шахматной доски 2×2 (чёрно-белые квадраты), tooltip: "Начальная расстановка"
2. **«Партия»** — текстовая кнопка с текстом "Партия" на тёмно-синем фоне
3. **«Анализ»** — текстовая кнопка с текстом "Анализ" на голубом фоне
4. **«Свернуть»** — круглая кнопка с горизонтальной линией (—)
5. **«Поверх всех окон»** — круглая кнопка с шахматной текстурой
6. **«Закрыть»** — круглая кнопка с крестиком (×)

### Нижний ряд (слева направо):
1. **«Ok»** — круглая кнопка с текстом "Ok" (только в Расширенном виде)
2. **«Меню»** — круглая кнопка с тремя горизонтальными линиями (☰)
3. **[Строка индикации хода]** — текстовое поле, НЕ кнопка
4. **«Предыдущий ход»** — круглая кнопка с двойной стрелкой влево («)
5. **«Следующий ход»** — круглая кнопка с двойной стрелкой вправо (»)
6. **«Удалить фигуру»** — круглая кнопка с крестиком в круге (⊗), голубой
7. **«Реверс»** — круглая кнопка с двумя круговыми стрелками (↻)
8. **«Изменить размер»** — точки в углу (⤡), диагональная стрелка

### Стиль кнопок (из макета):
- Круглые кнопки: диаметр ~32px, серебристый градиент фон, тёмная обводка
- Текстовые кнопки (Партия/Анализ): прямоугольные со скруглением, цвет как в палитре
- При замораживании: opacity 0.4, cursor: not-allowed
- При наведении: лёгкая подсветка (brightness 1.15)

```css
.castle-cell {
  background-color: #C0C0C0;
  background-image: repeating-linear-gradient(
    45deg,
    transparent,
    transparent 3px,

  );
}
```
